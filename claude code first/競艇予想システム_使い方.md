# 競艇予想システム 使い方

ファイルの場所: `/Users/nakajimataku/claude code second/`

---

## 基本コマンド

```bash
cd "/Users/nakajimataku/claude code second"
```

---

## コマンド一覧

### システム確認
```bash
python3 cli.py check
```
設定・API接続・予想エンジン・X認証が全部正常か一括確認。

---

### 今日のレース一覧
```bash
python3 cli.py today
```
全会場のレース一覧と投稿タイミングを表示。

```bash
python3 cli.py today --venue 13
```
特定会場のみ表示（13=尼崎）。

---

### レース予想プレビュー（投稿なし）
```bash
python3 cli.py race 13 5
```
尼崎5Rの予想とツイート内容を確認できる（Xには投稿されない）。

---

### レース予想して投稿
```bash
python3 cli.py race 13 5 --post
```
予想を確認後、`y` を入力すると実際にXへ投稿。

---

### スコア詳細内訳（予想根拠）
```bash
python3 cli.py scores 13 5
```
各艇のスコアが何点どの項目から来ているか表示。

---

### 生APIデータ確認
```bash
python3 cli.py data 13
```
尼崎の全レースデータをJSONで表示。

```bash
python3 cli.py data 13 --race 5
```
5Rのみ表示。

---

### 会場コード一覧
```bash
python3 cli.py venues
```
全24会場のコード番号を確認。

---

## 会場コードよく使うもの

| コード | 会場 |
|--------|------|
| 12 | 住之江 |
| 13 | 尼崎 |
| 14 | 鳴門 |
| 15 | 丸亀 |
| 22 | 福岡 |

---

## 自動投稿の設定変更（.env）

```
VENUE_CODES=13,22         ← 複数会場はカンマ区切り
POST_WINDOW_MIN=30        ← 締切30分前から投稿対象
POST_WINDOW_MAX=60        ← 締切60分前まで投稿対象
```
