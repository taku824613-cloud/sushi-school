寿司職人育成スクール・東京

3ヶ月で基礎から現場レベルまで。実践中心のカリキュラム。

Features

3つの特徴

寿司職人という職業の魅力を、短期間で身につけるための特徴。

素早く・深く・実践的に。
Feature 01
⏱

短期集中カリキュラム

3〜6ヶ月の短期集中で、包丁の握り方から握り寿司・巻物・前菜まで一連の基礎技術を体系的に習得します。働きながら通える時間割も用意しています。

Feature 02
🎌

実践中心の授業

デモンストレーションを見て終わりではなく、一人ひとりが実際に手を動かす実習が中心。プロの寿司職人が、現場で求められる所作とスピード感を丁寧に指導します。

Feature 03
↗

卒業後の就職サポート

国内外の寿司店・和食店とのネットワークを活かし、就職・転職をサポート。将来の独立開業についても、物件探しやメニュー設計などの相談が可能です。

Curriculum

カリキュラム

基礎から店舗運営まで。寿司職人として必要な要素を一つひとつ積み上げます。

未経験OK・日本語・英語対応
01
Module 01
寿司の基礎

米の研ぎ方・酢飯の合わせ方・包丁の持ち方など、寿司職人としての土台となる基本動作を徹底的に身につけます。

02
Module 02
魚のさばき方

鮮魚の選び方、三枚おろし、柵取り、保存方法まで。素材のポテンシャルを最大限に活かす「仕込み」を学びます。

03
Module 03
握り技術・盛り付け

握りの力加減、シャリとネタのバランス、スピード、盛り付けの美しさなど、「一貫の完成度」を左右する技術に集中して取り組みます。

04
Module 04
接客・ホスピタリティ

カウンターでの立ち居振る舞い、所作、言葉遣い、外国人ゲストへの対応など、高級寿司店にふさわしいサービスをロールプレイ形式で習得します。

05
Module 05
店舗運営・独立準備

メニュー構成、原価計算、仕入れ、スタッフマネジメント、ブランディング、海外出店のポイントなど、自分の店を持つための実践的な知識を学びます。

※詳細な時間割・開講スケジュールは無料説明会にてご案内します。

コース概要

3ヶ月 基礎集中 6ヶ月 プロコース 平日夜コース 週末コース 日本語・英語対応
Career

卒業後のキャリア

卒業後のキャリアは一つではありません。あなたの「なりたい寿司職人像」に合わせて伴走します。

国内就職・海外挑戦・独立開業
Path 01

寿司店・和食店への就職

都内の寿司店・和食店を中心に、卒業生向けの求人をご紹介。実技試験や面接対策も講師陣がサポートし、現場へのスムーズなデビューを後押しします。

Path 02

海外レストランでのキャリア

北米・ヨーロッパ・アジアなど、海外の寿司レストランと提携。ビザ取得や英語でのコミュニケーションについても、専門スタッフがご相談に応じます。

Path 03

自分の寿司店を開業

小さなカウンター寿司店から、海外でのポップアップ店舗まで。コンセプト設計やメニュー開発、数値管理の考え方まで、開業に必要なノウハウを共有します。
