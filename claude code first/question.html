💴 学費・ローン

費用について、
正直に答えます。

「思ったより高い」「ローンはあるの？」——よく聞かれる質問です。

Q 学費の総額はいくらですか？分割払いはできますか？ ＋

コースにより異なります。基礎集中コース（3ヶ月）は¥298,000、プロフェッショナルコース（6ヶ月）は¥498,000（いずれも税別・教材費・食材費含む）です。

一括払いのほか、2〜24回の分割払いに対応しています（金利は分割回数により異なります）。

※ 表示金額はプレースホルダーです。正式な料金は無料説明会でご案内します。
Q 教育ローンは利用できますか？審査が心配です。 ＋

はい、提携する教育ローン会社をご紹介できます。最大120回払い（最長10年）で月々の負担を抑えることが可能です。審査が不安な方でも、担当スタッフが申し込みをサポートしますのでお気軽にご相談ください。

  • 提携ローン会社への紹介状発行
  • 申し込み書類の確認サポート
  • 審査通過後の入学手続きフロー案内
Q 追加費用はかかりますか？後から請求されませんか？ ＋

学費に教材費・実習食材費・施設使用料をすべて含んでいます。入学後に予想外の請求が発生することはありません。

ただし、卒業認定試験の再受験料（1回につき¥15,000）、任意の海外研修プログラム参加費は別途かかります。

包丁は授業で使用するものを貸与しますが、自分の包丁を購入したい方には提携店の割引価格でご案内しています（任意）。
Q 奨学金や補助金の制度はありますか？ ＋

現在、独自の奨学金制度はありませんが、自治体の職業訓練補助や、ハローワークの教育訓練給付金の対象になるケースがあります。詳細はスタッフまでご相談ください。

また、早期申し込み割引（入学予定日の3ヶ月前までに申込で5%オフ）を実施しています。

🔪 実技・授業内容

「本当に短期間で
寿司が握れるようになるの？」

最もよく聞かれる疑問です。カリキュラムと実態を正直にお伝えします。

Q 本当に3ヶ月で寿司職人になれるのですか？ ＋

「なれる」と「なれない」の両方をお伝えします。3ヶ月で「現場デビューできる基礎技術」は確実に習得できます。ただし、10年修行した職人と同じレベルには当然なれません。

当アカデミーの3ヶ月コース修了者は、寿司店でのアルバイト・パート・アシスタント職人として採用されるレベルです。プロフェッショナルコース（6ヶ月）修了者は、即戦力の正規職人として就職できるレベルを目指します。

修行スタイルとの比較：従来の修行では「見て覚える」のが基本。当校では体系化された授業と毎日の反復練習で、3ヶ月が従来の1〜2年分の習得速度に相当します。
Q 実技の割合はどのくらいですか？座学中心ではないですか？ ＋

カリキュラム全体の約75%が実技授業です。座学（衛生管理・食品知識・経営基礎）は25%程度。「習うより慣れろ」を体系的に実現した構成です。

  • 午前：魚のさばき方・仕込み（実技）
  • 午後：握り・巻き物・装飾（実技）
  • 週1回：衛生管理・食文化・接客マナー（座学）
Q 料理が全くの未経験でも大丈夫ですか？ ＋

全く問題ありません。全受講生の約80%が料理未経験からスタートしています。包丁の持ち方・砥ぎ方から始まり、段階的にレベルアップするカリキュラムです。むしろ「変な癖がない分、吸収が早い」という声もあります。

Q 1クラス何人ですか？マンツーマンに近い指導を受けられますか？ ＋

1クラスの定員は最大8名です。講師1名に対して生徒8名以内を徹底しており、個別のフィードバックが受けられる体制を維持しています。大手スクールの20〜30名クラスと比べて、圧倒的に手厚い指導が特徴です。

🐟 魚・食材

魚は毎日触れますか？
食材費は別途かかりますか？

実習で使う魚や食材についての正直な情報をお届けします。

Q 授業で使う魚の食材費は別途かかりますか？ ＋

食材費はすべて学費に含まれています。授業で使用するマグロ・サーモン・ヒラメ・サバ・イカなどの魚介類、シャリ用の酢・米、のりなどの追加費用は一切かかりません。

3ヶ月コースで使用する魚の総量は約120〜150kg。毎日本物の魚に触れる実習環境を用意しています。
Q 毎日魚を触ることはできますか？どんな魚を扱いますか？ ＋

はい、毎授業日に生魚を扱います。基礎コースでは以下の魚介類を順番に習得します：

  • マグロ（赤身・中トロ・大トロの柵取り）
  • サーモン（ノルウェー産 / 国産ます）
  • ヒラメ・カレイ（五枚おろし）
  • サバ・アジ・イワシ（光物の仕事）
  • イカ・タコ・エビ・貝類
  • プロコースでは加えてフグ・クエ・天然ものなど高級魚も扱います
Q 授業で捌いた魚や握った寿司は自分で食べられますか？ ＋

はい、授業中に握った寿司・調理した料理は、その日の授業終わりにクラス全員でいただきます。「食べながら評価する」ことも重要な学習です。

食べきれない量になることもありますが、持ち帰り容器を用意していますので持ち帰りも可能です（衛生上、当日中にお召し上がりください）。

受講生からは「毎日の賄いが楽しみで、食べながら自分の技術の課題を実感できる」という声が多いです。
Q 魚の臭いが苦手でも大丈夫ですか？ ＋

鮮度の高い魚は「生臭い」においがほとんどありません。当校では豊洲市場から仕入れた鮮魚のみを使用しており、適切な保管・処理の仕方も授業の一部です。「最初は苦手だったが慣れた」という受講生が多いです。まずは無料体験説明会で実際に体験してみてください。

Q 卒業後の就職サポートはどのような内容ですか？ ＋

就職サポートは卒業後2年間、無料で利用できます。内容は以下の通りです：

  • 国内外200店舗以上の提携寿司店・和食店への求人紹介
  • 履歴書・職務経歴書の添削・日英両語対応
  • 模擬面接・技術実技試験の対策練習
  • 海外就職希望者への現地スタッフ紹介
  • 独立開業希望者への物件・資金調達相談
Q 卒業後の平均年収はいくらくらいですか？ ＋

卒業後の年収は就職先・地域・国によって大きく異なりますが、国内寿司店勤務の場合は1年目で280〜380万円、3〜5年後に400〜600万円が一般的です。

海外（北米・欧州・中東）では日本の1.5〜3倍の年収水準になるケースも多く、ニューヨーク・ロンドン勤務では年収600〜1,200万円相当の報酬を得る卒業生もいます。

※ 上記数値は参考値です。実際の就職先・年収事例は無料説明会でご案内します。
Q 卒業後に独立開業することはできますか？ ＋

はい。プロフェッショナルコースでは店舗運営・原価計算・メニュー開発・衛生管理まで学べるカリキュラムを設けており、独立を視野に入れた受講生も多いです。

卒業後のサポートとして、以下の相談に対応しています：

  • 物件探しのアドバイス（提携不動産会社の紹介）
  • 融資・資金調達の相談（日本政策金融公庫への紹介状）
  • メニュー設計・原価率計算のコンサルティング
  • 保健所への営業許可申請の手順説明

🌏 外国人・ビザ

外国人でも
入学できますか？

全受講生の約40%が外国人。グローバルな環境が当校の強みです。

Q 外国人でも入学できますか？英語で授業を受けられますか？ ＋

はい、外国人の方を積極的に受け入れています。英語での授業対応・英語テキスト提供が可能です。中国語・スペイン語については翻訳テキストのみ対応しています。

  • 英語対応可能な講師常駐
  • 英語版カリキュラムテキスト提供
  • 英語・日本語の両言語で卒業証明書発行
  • 入学手続き書類の英語サポート
Q 在留資格（ビザ）の取得サポートはありますか？ ＋

はい、ビザ取得のサポートを行っています。「技能ビザ」「特定技能ビザ」「留学ビザ」など、目的・状況に応じたビザの相談に対応しています。

提携する行政書士・入国管理局手続き専門家をご紹介することも可能です。

ビザの種類や取得可否は国籍・在留状況により異なります。詳細は個別相談（無料）にてご確認ください。
Q 日本語がほとんど話せなくても授業についていけますか？ ＋

実技中心の授業ですので、日本語が全く話せない方でも基本的な技術習得は可能です。ただし、衛生管理や食品知識の座学については英語テキストと通訳サポートを利用していただきます。日常会話レベルの英語があれば問題なく受講できます。

日本語学習に興味がある方向けに、基礎日本語の自学習資料も提供しています（任意）。

Q 海外から日本に来て入学したいのですが、住居の相談はできますか？ ＋

はい、提携するシェアハウス・学生寮のご紹介が可能です。東京都内・アクセス良好エリアで、月額50,000〜80,000円程度の物件を中心に案内しています。住居探しの不安を解消するサポートパッケージ（別途有料）もご用意しています。

Q JSSAに向いていない人はどんな人ですか？ ＋

正直にお伝えします。以下のような方には他の選択肢をお勧めします：

  • 「3ヶ月で一人前の職人になれる」という過度な期待がある方
  • 週2〜3日の通学で十分と考えている方（集中カリキュラムのため毎日通学が原則）
  • 調理の世界に全く興味がなく、とりあえず資格が欲しいだけの方
向いているのは「本気でプロを目指したい方」「海外で活躍したい方」「短期集中で確実に技術習得したい方」です。
❓ その他

その他の
よくある質問

入学前に気になる細かな疑問にお答えします。

Q 仕事をしながら通うことはできますか？夜間・週末コースはありますか？ ＋

はい、平日夜間コース（18:30〜21:30）と週末コース（土日 10:00〜16:00）をご用意しています。現在の仕事を続けながら寿司職人を目指すことが可能です。

  • 平日昼コース：月〜金 9:00〜16:00（3ヶ月修了）
  • 平日夜間コース：月〜木 18:30〜21:30（5ヶ月修了）
  • 週末コース：土日 10:00〜16:00（6ヶ月修了）
Q 年齢制限はありますか？50代・60代でも入学できますか？ ＋

入学に年齢制限はありません。18歳〜65歳まで幅広い年代が在籍しています。定年後のセカンドキャリアとして入学する方、子育て後の社会復帰として学ぶ方など、多様なバックグラウンドの方が活躍しています。

Q 入学前に体験授業はできますか？ ＋

はい、無料の体験説明会（2時間）を毎週実施しています。実際に包丁を握って魚をさばく体験ができます。オンライン説明会も対応しています。

申し込みはトップページのフォームから。2営業日以内にご連絡します。

Q 途中でやめた場合、学費は返金されますか？ ＋

入学後8日以内のキャンセルについては全額返金対応しています（特定商取引法に基づくクーリングオフ）。それ以降の中途退学については、受講済み期間分を差し引いた残額の50%を返金するポリシーです。

ご入学前に契約書の返金ポリシーをしっかりご確認いただき、納得いただいてからの入学をお願いしています。
Q 卒業証書・資格はもらえますか？履歴書に書けますか？ ＋

修了試験合格者には「日本寿司職人アカデミー修了証（日英二言語）」を発行します。国家資格ではありませんが、国内外の寿司店への就職活動において実績として活用いただけます。

また、食品衛生責任者資格の取得支援も実施しています（東京都知事登録）。

Q スクールはどこにありますか？アクセスは便利ですか？ ＋

東京都内（詳細住所は確定後更新予定）。最寄り駅から徒歩5分以内のアクセスしやすい立地を予定しています。

まだ疑問が解決しない方へ。個別相談（無料）をご利用ください。

電話・メール・オンラインで、入学担当スタッフが一対一でお答えします。
