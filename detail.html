Curriculum

カリキュラム

4つのフェーズで、基礎から現場実務レベルまで段階的に積み上げます。
2ヶ月集中コースは週5日・全8週間、週末特訓コースは週1回・12ヶ月で同カリキュラムを修了します。

01
入門期 / Phase 01
基本技術の習得
2ヶ月コース：第1〜2週目　|　週末コース：1〜3ヶ月目
寿司の基礎 ＆ 魚のさばき方

米の研ぎ方・酢飯の合わせ方・包丁の基本から始まり、鮮魚の目利き・三枚おろし・柵取りまで、職人としての土台となる技術を反復習得します。

酢飯の調合・炊飯・温度管理
包丁の研ぎ方・柄の持ち方・衛生管理
マグロ・ハマチ・サーモン・アジ・エビのさばき方
食材の熟成・塩締めの基礎知識
米の品種選び・洗米・浸水・炊飯の基礎から合わせ酢の調合と温度管理まで。包丁の研ぎ方と柄の持ち方、まな板の使い方を含め衛生管理の基礎も学びます。魚は主要魚種（まぐろ・さば・あじ・ひらめ・えび等）を実習し、市場での仕入れ目利き体験も実施します。
02
理解期 / Phase 02
江戸前伝統技法
2ヶ月コース：第3〜4週目　|　週末コース：4〜6ヶ月目
江戸前の塩〆・酢〆技法をマスター

江戸前寿司の真髄である「塩〆・酢〆」の技法を集中的に習得。サバ・コハダ・イナダなど繊細な魚の処理法と、刺身・棒寿司の仕上げ技術を身につけます。

サバ・コハダ・イナダの塩〆・酢〆
刺身の引き方・棒寿司の製作
食材衛生管理・原価計算の基礎
握りの基本型（本手返し）の反復練習
江戸前寿司の伝統技法である塩〆・酢〆を徹底的に習得します。各魚種の特性に応じた締め時間・塩加減・酢の濃度調整を実習。合わせて原価計算の基礎知識と食材コスト管理を学び、現場で即使えるビジネス感覚も養います。
03
体得期 / Phase 03
高難度魚類・握りの完成度
2ヶ月コース：第5〜6週目　|　週末コース：7〜9ヶ月目
難易度の高い白身魚と握りの精度向上

タイ・ヒラメ・アナゴなど難易度の高い白身魚の処理を習得。形の美しさとスピードを両立させた握りの精度を高め、軍艦巻き・手巻き・巻き物も総合実習します。

タイ・ヒラメ・アナゴの捌き方と握り
シャリとネタのバランス・握りのスピード向上
軍艦巻き・手巻き・細巻き・太巻き
盛り付けの美意識・彩りの構成
江戸前の本手返しから握りの基本型まで反復練習を重ね、形状・美しさにこだわった一貫に仕上げます。20種以上の魚類を通じて各素材の特性を体で覚え、スピードトレーニングで現場水準の生産性も習得します。
04
飛躍期 / Phase 04
現場実務・接客応用
2ヶ月コース：第7〜8週目　|　週末コース：10〜12ヶ月目
カウンター対応・現場レベルの総仕上げ

現場水準のスピード感・カウンター対応能力を養い、コース料理の全工程を通じて実務経験を積みます。外国人ゲストへの英語対応や、活だこの処理技法も習得します。

カウンター研修・ロールプレイ実習
日本語・英語での接客応対
活だこ処理・野菜装飾技法
コース料理の全工程シミュレーション
日本語・英語双方のカウンター応対ロールプレイを実施。お客様との間合い、ペアリング提案、アレルギー対応のプロトコルを習得します。修了試験では実際に握りのスピード・精度・接客の3項目を評価します。
05
独立準備 / Phase 05
店舗運営・開業ゼミ
週末特訓コース 選択受講
自分の店を持つための実践知識

メニュー構成・原価計算・仕入れ交渉・スタッフマネジメント・ブランディングなど、開業に必要なすべてのノウハウを体系的に学びます。海外出店を目指す方向けの内容も含みます。

メニュー構成・原価計算・仕入れ交渉
店舗ブランディング・SNS活用
スタッフ採用・マネジメント基礎
海外出店・インバウンド対応のポイント

Career

卒業後のキャリア

卒業後のキャリアは一つではありません。あなたの「なりたい寿司職人像」に合わせて伴走します。

Path 01

寿司店・和食店への就職

都内の寿司店・和食店を中心に、修了生向けの求人情報をご提供できるよう提携先を開拓しています。実技試験や面接対策も講師陣がサポートし、現場へのスムーズなデビューを後押しします。

初年度収入目安
年収 350〜550万円（都内寿司店 正社員）
Path 02

海外レストランでのキャリア

北米・ヨーロッパ・アジアなど、海外の寿司レストランと提携。ビザ取得や英語コミュニケーションについても、専門スタッフがご相談に応じます。

初年度収入目安
年収 500〜1,000万円+（北米・欧州・アジア圏）
Path 03

自分の寿司店を開業

小さなカウンター寿司店から海外でのポップアップ店舗まで。コンセプト設計・メニュー開発・数値管理の考え方まで、開業に必要なノウハウを共有します。

開業2〜3年後収入目安
年収 700万円〜（店舗規模により変動）
